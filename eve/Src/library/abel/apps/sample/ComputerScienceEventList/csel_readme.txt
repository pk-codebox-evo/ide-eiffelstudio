CSEL Directory structure:
- conf: configuration file: it will go into the cgi-bin dir in the deployed app
- informatics_events: CSEL sources
- web: html and javascript
- web/css: css
- web/data: CSEL data files (resulting from binary serialization)
- web/data/backup: backup dir for data files
- web/images: images
- web/sessions: session files (resulting from binary serialization) 
- webex: web framework
